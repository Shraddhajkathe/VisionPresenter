# VisionPresenter

**VisionPresenter: A Touchless Presentation Control System Using Computer Vision**

## Overview

VisionPresenter is a desktop application that allows users to control presentations using hand gestures detected through a webcam. The system is designed to provide a touchless presentation experience with features such as slide navigation, annotation, zoom, volume control, and voice feedback.
